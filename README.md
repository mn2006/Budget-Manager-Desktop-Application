# Budget Manager Desktop Application

Budget Manager is a smart desktop application developed using JavaFX and SQLite to help users efficiently manage their personal finances. The system allows users to create budget cycles, track expenses, categorize transactions, monitor spending progress, and visualize financial data through interactive charts.

The application follows the MVC architecture and applies software engineering principles such as modularity, encapsulation, and maintainability. It provides an intuitive and user-friendly interface that helps users control spending habits and improve financial awareness.

## Features

* Budget cycle management
* Expense tracking and categorization
* Real-time spending analysis
* Interactive pie chart visualization
* Budget warning system
* Transaction history management
* Edit and delete expense records

## Technologies Used

* Java 17
* JavaFX
* SQLite
* Maven

This project was developed as part of the Introduction to Software Engineering course at Cairo University.
